# Pomodoro Timer

> A simple timer based on the standard rules provided by The Pomodoro Technique

## Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:8080
yarn start
```
The app should be running on:
- <https://localhost:3000>
